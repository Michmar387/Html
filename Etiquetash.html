HTML

¿Que es?

Es un lenguaje de marcado que se utiliza para el desarrollo de paginas de internet. Se trata
de las siglas que corresponden a Hipertexto Markup language, es decir, lenguaje de marcas de
Hypertexto. No obstante, este tipo de definiciones no nos dicen mucho porque la definicion
es tecnica. Para algunas personas al leer esto, piensa que HTML incluye el diseno grafico de las paginas
web, sin embargo, eso no es cierto ya que HTML solo sirve para indicar como va ordenado el contenido de una
pagina web. Esto lo hace por medio de las marcas de hypertexto las cuales son etiquetas
conocidas en ingles como tags. Las etiquetas se caracterizan por que van dentro de los caracteres
menor que< y mayor que>, como en este ejemplo aunque eso no es todo debido a que
deben tener un inicio y un final. El inicio es una etiqueta, en otras palabras lleva los
dos caracteres que se mencionaron anteriormente y dentro de estos se encuentra la etiqueta
correspondiente y para el final existe una diferencia, en la etiqueta debemos escribir
una diagonal /.
